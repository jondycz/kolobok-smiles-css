KOLOBOK Smiles CSS

An unofficial CSS library for embedding KOLOBOK Smiles in your web projects using simple class names. Click any smile to copy its HTML snippet.

Usage

Include the CSS file in your HTML, then use a <span> element with the appropriate classes:

<link rel="stylesheet" href="kolobok.min.css">

<span class="kolobok acute"></span>
<span class="kolobok dance"></span>
<span class="kolobok kiss"></span>
